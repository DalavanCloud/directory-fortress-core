This package contains samples that show how the Fortress APIs can be used. The samples use JUnit and provide the following.

Samples

  1. {@link com.jts.fortress.samples.AccessMgrSample} demonstrates {@code com.jts.fortress.AccessMgr} APIs
  2. {@link com.jts.fortress.samples.AllSamplesJUnitTest} will execute all tests in their proper order
  3. {@link com.jts.fortress.samples.CreatePermOrgHierarchySample} demonstrates {@code com.jts.fortress.DelegatedAdminMgr} APis to create Perm organizational unit hierarchies
  4. {@link com.jts.fortress.samples.CreatePermOrgSample} demonstrates {@code com.jts.fortress.DelegatedAdminMgr} APis to create a Perm organizational unit node
  5. {@link com.jts.fortress.samples.CreatePermSample} demonstrates {@code com.jts.fortress.AdminMgr} to create RBAC permissions
  6. {@link com.jts.fortress.samples.CreateRoleHierarchySample} demonstrates {@code com.jts.fortress.AdminMgr} to create RBAC Role hierarchies
  7. {@link com.jts.fortress.samples.CreateRoleSample} demonstrates {@code com.jts.fortress.AdminMgr} to create RBAC Roles and assign constraints
  8. {@link com.jts.fortress.samples.CreateSessionSample} demonstrates various ways to create a session using {@code com.jts.fortress.AccessMgr} APIs
  9. {@link com.jts.fortress.samples.CreateUserOrgHierarchySample} demonstrates {@code com.jts.fortress.DelegatedAdminMgr} APis to create User organizational unit hierarchies
  10. {@link com.jts.fortress.samples.CreateUserOrgSample} demonstrates {@code com.jts.fortress.DelegatedAdminMgr} APis to create a User organizational unit node
  11. {@link com.jts.fortress.samples.CreateUserRoleSample} demonstrates {@code com.jts.fortress.AdminMgr} APIs to assign Roles to Users
  12. {@link com.jts.fortress.samples.CreateUserSample} demonstrates {@code com.jts.fortress.AdminMgr} APIs to create User object nodes

The com.jts.fortress.samples package provides examples on proper usage of Fortress APIs, Each sample contains javadoc and JUnit test case that describes the use case and function for a particular entity. For more info, check out the javadoc pertaining to each sample.
