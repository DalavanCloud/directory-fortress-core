This package contains APIs to perform CRUD on fortress container objects in ldap.

The org.openldap.fortress.ldap.container package provides apis to add and remove ldap OrganizationalUnit entries. These entries are called 'containers' and are used to organize related fortress nodes. An example container is ou=People which will be placed as a child node of the directory's ldap suffix, i.e. dc=example,dc=com. The container is used to store subentries, i.e. inetOrgPerson. These container APIs are used by buildup/teardown processes.
