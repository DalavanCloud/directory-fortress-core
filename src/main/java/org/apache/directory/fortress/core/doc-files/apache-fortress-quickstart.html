Overview

The documents linked by this guide provides a how-to guide for installing Apache Fortress software.

Quickstart Guides

The following topics are covered:
  1. Setup Apache Directory Server
  2. Build Apache Fortress Core
  3. Build Apache Fortress Realm
  4. Setup Apache Tomcat Web Server
  5. Build Apache Fortress Rest
  6. Build Apache Fortress Web

Prerequisites

  1. Java 7 (or greater) SDK
  2. Apache Ant 1.8 (or greater)
  3. Apache Maven 3
  4. Firefox Web Browser

Getting Started - Generating the tutorial's how-to guide

This tutorial uses javadoc that is contained within the apache-fortress-core source archive. To generate, follow these instructions:
  1. Download the apache-fortress-core source bundle from Apache Git repo.
  2. Extract bundle
  3. Set JAVA_HOME
  4. Set M2_HOME
  5. Run this command from the root package: 
                        
    
$ $M2_HOME/bin/mvn javadoc:javadoc
  6. Open the help document: /*package home*/target/site/apidocs/org/apache/directory/fortress/core/doc-files/apache-fortress-quickstart.html
  7. Complete the desired sections under Quickstart Guides section

Infrastructure Installed

  1. Apache Directory Server
  2. Apache Directory Studio
  3. Apache Fortress
  4. Apache Tomcat Servlet Container
