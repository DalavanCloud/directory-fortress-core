    • Build Apache Fortress Realm

    This section describes how to install The Fortress Realm Java EE Policy Enforcement Component

    1. Clone the Apache Fortress Realm Git repo: 
                      
      
# git clone https://git-wip-us.apache.org/repos/asf/directory-fortress-realm.git
    2. Set JAVA_HOME: 
                      
      
# export JAVA_HOME=/opt/jdk1.7.0_10/
    3. Change directory: 
                      
      
# cd directory-fortress-realm/
    4. Set M2_HOME: 
                      
      
# export M2_HOME=/usr/local/maven-version

    5. Build fortress realm with maven and install to local repo. The fortress rest [Build Apache Fortress Rest] and fortress web [Build Apache Fortress Web] are both dependent on the fortress realm. 
                          
      
# $M2_HOME/bin/mvn clean install
