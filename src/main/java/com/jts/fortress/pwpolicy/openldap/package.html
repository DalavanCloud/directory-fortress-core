This package provides CRUD for policy and interrogation of password control objects stored in OpenLDAP.

The com.jts.fortress.pwpolicy.openldap package uses {@link com.jts.fortress.pwpolicy.PswdPolicy PswdPolicy} to store and retrieve data elements in OpenLDAP ldap object class pwdPolicy. In addition this package contains processing to interrogate the OpenLDAP password policy control 1.3.6.1.4.1.42.2.27.8.5.1. All apis contained within this package are for fortress use only. See the corresponding javadoc contained with this package for more info.
