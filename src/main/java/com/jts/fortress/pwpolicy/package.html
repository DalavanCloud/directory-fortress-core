This package provides CRUD to ldap password policy entities and interrogation of password control objects.

The com.jts.fortress.pwpolicy package data entity, see {@link com.jts.fortress.pwpolicy.PswdPolicy PswdPolicy}, may be manipulated by outside programs but the apis contained within this package are for fortress use only. See the corresponding javadoc contained with this package for more info. This package stores and retrieves data elements in OpenLDAP ldap object class pwdPolicy. In addition this package contains processing to interrogate the OpenLDAP password policy control 1.3.6.1.4.1.42.2.27.8.5.1.
