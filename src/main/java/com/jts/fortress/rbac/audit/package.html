This package provides data entities and APIs to interrogate OpenLDAP's slapd access log to obtain fortress audit trail and historical events.

The com.jts.fortress.rbac.audit package data entities may be manipulated by outside programs but the apis contained within this package are for fortress use only. The entities include {@link com.jts.fortress.audit.AuthZ AuthZ}, {@link com.jts.fortress.audit.Bind Bind}, {@link com.jts.fortress.audit.Mod Mod} and {@link com.jts.fortress.audit.UserAudit UserAudit} which is used to input search criteria. See the corresponding javadoc contained with this package for more info.

Audit Interrogator

Provides an OpenLDAP access log retrieval mechanism that enables security event monitoring.
  1. Authentication events:
  2. Session enablement events
  3. Authorization events
  4. Entity mods and deletes

All events include Fortress context, see {@link FortEntity}.

The following APIs generate events subsequently stored in this access log:

  • {@link com.jts.fortress.AccessMgr}
  • {@link com.jts.fortress.AdminMgr}
  • {@link com.jts.fortress.AdminMgr}
  • {@link DelegatedAdminMgr}
  • {@link com.jts.fortress.configuration.ConfigMgr}
  • {@link PswdPolicyMgr}

The following reports are supported using search input: {@link com.jts.fortress.rbac.audit.UserAudit}

  • User Authentications: List&lt;{@link com.jts.fortress.rbac.audit.Bind}&gt; {@link AuditMgr#scom.jts.fortress.rbac.audittress.audit.UserAudit)}
  • Invalid Users AuthN: List&lt;{@licom.jts.fortress.rbac.auditdit.Bind}&gt; {@link AuditMgr#searchAuthNInvalid(com.jts.fortress.rbac.audit.UserAudit)}
  • User Authorizations 1: List&lt;{@licom.jts.fortress.rbac.auditdit.AuthZ}&gt; {@link AuditMgr#getAuthZs(com.jts.fortress.rbac.audit.UserAudit)}
  • User Authorizations 2: List&lt;{@link com.jts.fortress.rbac.audit.AuthZ}&gt; {@link AuditMgr#secom.jts.fortress.rbac.audittress.audit.UserAudit)}
  • User Session Activations: List&lt;{@licom.jts.fortress.rbac.auditdit.Mod}&gt; {@link AuditMgr#searchMods(com.jts.fortress.rbac.audit.UserAudit)}
  • Entity Modifications: List&lt;{@link com.jts.fortress.rbac.audit.Mod}&gt; {@link AuditMgr#searccom.jts.fortress.rbac.audittress.audit.UserAudit)}
