This package uses JGraphT to provide hierarchical functionality that is reused within several other packages in the fortress system.

The com.jts.fortress.hier package implements a simple directed graph data structure that is reused across the various hierarchical subsystems in this system. The simple digraph functions utilize a 3rd party open source package, see JGraphT to provide this capability. This package also provides the ability to store the parent-child relationships in ldap object class oamHier. See the corresponding javadoc contained with this package for more info.
